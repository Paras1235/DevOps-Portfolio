\# Flask Skill Tracker \& Portfolio Dashboard



A dynamic web application built to track technical skills, designed as a foundation for end-to-end DevOps and Data Engineering practice.



\## 🚀 Project Overview

This project is a CRUD-based (Create, Read, Update, Delete) web application that allows users to manage a list of technical skills. While the app is simple, it is built with a \*\*production-first mindset\*\*, serving as the primary application for my CI/CD pipeline experiments.



\## 🛠️ Tech Stack

\* \*\*Backend:\*\* Python 3 \& Flask Framework

\* \*\*Frontend:\*\* HTML5, CSS3, and Jinja2 Templating

\* \*\*Version Control:\*\* Git \& GitHub

\* \*\*Infrastructure Target:\*\* Docker \& Jenkins (In Progress)



\## 📋 Features

\* \*\*Add Skills:\*\* Input field to dynamically update the skill list.

\* \*\*Clear Data:\*\* One-click functionality to reset the dashboard.

\* \*\*Clean UI:\*\* Professional card-based layout with responsive styling.



\## 📈 DevOps Roadmap

This project is part of a larger learning path to master \*\*Data Engineering\*\* infrastructure:

1\. \*\*Containerization:\*\* Wrap the app in a Docker container.

2\. \*\*Automation:\*\* Set up a Jenkins pipeline for automated testing.

3\. \*\*Deployment:\*\* Host the container on an Ubuntu-based Home Lab.
